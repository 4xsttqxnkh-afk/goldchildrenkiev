# GoldChildrenKiev - Starter

## How to run locally

1. Install dependencies:
   npm install
2. Run dev server:
   npm run dev
3. Build:
   npm run build

## Deploy to Vercel
1. Create a Git repo and push this project.
2. Sign in to Vercel and import the repo.
3. Set framework to 'Vite' (auto-detected) and deploy.
